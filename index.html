How to get a Google Client ID
  1. Go to console.cloud.google.com
  2. Create a new project (e.g. TV Tracker)
  3. Enable the Google Calendar API (APIs & Services → Library)
  4. Go to Credentials → Create OAuth Client ID
  5. Type: Web application
  6. Add your GitHub Pages URL as an Authorized JavaScript origin
    (e.g. https://yourname.github.io)
  7. Copy the Client ID and paste it above, then tap Save Settings
