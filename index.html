How to get a Google Client ID
  1. Go to console.cloud.google.com
  2. Create a project named TV Tracker
  3. Enable the Google Calendar API
  4. Go to Credentials → Create OAuth Client ID
  5. Type: Web application
  6. Add your GitHub Pages URL as Authorized JavaScript origin
  7. Copy the Client ID, paste above, tap Save Settings
